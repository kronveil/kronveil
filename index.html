Kronveil

AI-powered infrastructure observability agent

Chart v0.1.0 Helm 3.x Kubernetes 1.26+ Apache 2.0

Quick Install

1
Add the Helm repository
helm repo add kronveil https://kronveil.github.io/kronveil
helm repo update
2
Install Kronveil
helm install kronveil kronveil/kronveil \
  --namespace kronveil \
  --create-namespace \
  --set bedrock.region=us-east-1 \
  --set collectors.kubernetes.enabled=true
3
Verify
kubectl get pods -n kronveil
kubectl port-forward svc/kronveil-agent 8080:8080 -n kronveil
