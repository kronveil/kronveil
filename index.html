// Architecture

Designed for
enterprise scale.

Every layer purpose-built for high-throughput, mission-critical environments. Kafka as the backbone. Bedrock as the brain.

01

Collectors Layer

Lightweight agents deployed per cluster/cloud. Emit structured events to the Kafka backbone at 10M+ eps.

02

Kafka Event Bus

Multi-region Confluent Kafka clusters with Cluster Linking. The central nervous system of Kronveil.

03

Intelligence Engine

AWS Bedrock LLMs for root-cause analysis, anomaly detection ML, and capacity forecasting models.

04

Responders & Integrations

Autonomous remediation, ChatOps via Slack, GitOps triggers, and PagerDuty escalations.

// Quick Start

Up and running
in minutes.

Deploy Kronveil into any Kubernetes cluster with a single Helm command.

Helm
kubectl
Docker
# Add the Kronveil Helm repo
$ helm repo add kronveil https://kronveil.github.io/kronveil
$ helm repo update

# Install with AWS Bedrock + Kubernetes collectors
$ helm install kronveil kronveil/kronveil \
  --namespace kronveil \
  --create-namespace \
  --set bedrock.region=us-east-1 \
  --set collectors.kubernetes.enabled=true \
  --set collectors.kafka.enabled=true

✓ Release "kronveil" deployed to namespace "kronveil"
✓ Dashboard: http://localhost:8080 (after port-forward)
