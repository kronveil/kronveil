# Contributing to Kronveil

Thank you for your interest in contributing! Kronveil is built by platform engineers, for platform engineers — and we welcome contributions of all kinds.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/kronveil.git`
3. Set up dev environment: `make dev-setup`
4. Create a branch: `git checkout -b feature/your-feature-name`
5. Make your changes, add tests
6. Submit a Pull Request

## Areas We Need Help

- 🔌 New collector integrations (Azure Monitor, GCP Ops, Datadog)
- 🧠 LLM prompt engineering for better root-cause analysis
- 📊 Dashboard widgets and visualizations
- 📝 Documentation improvements
- 🌐 Internationalization (i18n)

## Code Standards

- Go 1.21+ for backend components
- Python 3.11+ for ML/intelligence components  
- TypeScript/React for dashboard
- All PRs must include tests
- Follow existing code style
